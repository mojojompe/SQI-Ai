# ML Models Directory

This directory is meant to store the compiled machine learning models used by the Django interface:

1. `house_price_model.pkl`
2. `city_label_encoder.pkl`
3. `city_classes.pkl`
4. `loan_approver_model.pkl`
5. `loan_features.pkl`

**To generate these files:**
1. Open and run all cells in `House_Price_Prediction_Model.ipynb`.
2. Open and run all cells in `Loan_Approver_model.ipynb`.
3. Copy the generated `.pkl` files from those directories into this `ml_models` directory.
